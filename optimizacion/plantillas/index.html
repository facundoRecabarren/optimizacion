OPTIMIZATION API

Introduction

Welcome, with this API you can optimize the space usage of your trucks trying to reduce the spent fuel.
Distributing the packages among the trucks maximizing the volume usage. Restrictions: Each package should be accommodated in a single truck. The total weight of the packages in a single truck must not overweight the truck capacity.

Example

The API receives a JSON inside the POST body request: 

                {{jsonStyle1}}
After the request is processed it will response: 
                {{jsonStyle2}}

Information fields:
For each truck you have: Capacidad, nombre, identificador and patente.
Each marker(destination of the packages) has: id, lat(latitude), lon(longitude) and a list of packages for each destination. Each Package: id, description, weight and volume.
